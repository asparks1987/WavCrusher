Built for cold audio storage

Your WAV library is valuable. It does not have to stay enormous.

WAV is dependable, but it is brutally inefficient for finished collections that rarely need instant playback. WavCrusher is made for archived albums, exported stems, field recordings, podcast masters, sample-source folders, oral histories, and personal lossless music collections.

Great fit

  • Large WAV trees you want to keep lossless.
  • Finished projects that are mostly sitting on drives.
  • Collections where metadata chunks and exact wrapper bytes matter.
  • Offline archives that need readable manifests and recovery instructions.

Use another copy for

  • Daily playback libraries that need instant app compatibility.
  • Active DAW sessions still being edited.
  • Lossy delivery, streaming, normalization, or mastering changes.

Lossless with receipts

Every Verified item has to survive a real restore.

01

Hash the original WAV

WavCrusher streams SHA-256 over the full source file, not only decoded samples.

02

Encode losslessly

The pinned WavPack encoder creates a standard `.wv` file using the archival profile.

03

Restore in scratch space

The archive is decoded away from the source so verification never depends on wishful thinking.

04

Compare whole-file SHA-256

Only matching length and full-file hash allow the item to become Verified.
